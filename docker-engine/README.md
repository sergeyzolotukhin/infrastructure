#### Main commands

- run an environment
```shell
vagrant up
```

- Run a Docker containers
```shell
ssh root@192.168.56.2 dev-env up
```

- Stop a Docker containers
```shell
ssh root@192.168.56.2 dev-env down
```

- Stop the environment.
```shell
vagrant suspend  
```

#### Support commands

- SSH to the environment
```shell
ssh root@192.168.56.2
```
